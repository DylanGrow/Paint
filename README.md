# Tomás Estrada - Painting & Construction Services

Professional bilingual (English/Spanish) landing page for painting and construction services in North Carolina.

## 🚀 Features

- 🌐 Bilingual toggle (ES/EN) with localStorage persistence
- 🎠 Auto-rotating hero carousel (5s intervals)
- 🎤 Voice chat widget with Web Speech API + LibreTranslate/MyMemory fallback
- 🌤️ 3-day weather via wttr.in (no API key required)
- 💬 10 quick bilingual phrase buttons with TTS support
- 📞 Sticky Call/Text buttons with pre-filled SMS
- ♿ WCAG 2.1 AA accessible, semantic HTML, ARIA labels
- 📱 Mobile-first responsive design
- ⚡ Performance optimized: lazy images, minified CSS/JS, no frameworks
