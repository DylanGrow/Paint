# Tomás Estrada - Painting & Construction Services

Professional bilingual landing page optimized for mobile.

## 🚀 Quick Deploy

1. Create a new GitHub repository
2. Upload these files:
   - `index.html`
   - `css/style.css`
   - `js/app.js`
   - `js/voice.js`
   - `js/weather.js`
3. Go to Settings → Pages → Source: main branch → Save
4. Your site will be live at `https://yourusername.github.io/repo-name`
